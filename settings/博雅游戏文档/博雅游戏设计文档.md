# 《博雅：千禧回响》游戏设计文档

## 项目概述
- **引擎**: Unreal Engine 5
- **风格**: 千禧年复古Low-Poly (PS2/Dreamcast时代)
- **主题**: 剧情向恐怖/悬疑校园探索
- **背景**: 湛江博雅学校（改编）

---

## 一、核心视觉风格 (UE5实现)

### 1.1 渲染技术方案

#### 后处理材质设置
```
Post Process Volume 配置:
├── Dithering: Ordered/Bayer 4x4 (模拟16位色深)
├── Pixelization: 0.5-0.7 (降低分辨率感)
├── Scanlines: 可选开关 (CRT效果)
├── Color Grading:
│   ├── Saturation: 0.8-0.9 (略微降低饱和度)
│   ├── Contrast: 1.1-1.2 (提高对比度)
│   └── Shadow Tint: 偏蓝/偏绿 (千禧年阴郁感)
└── Vignette: 中度暗角
```

#### 材质系统
- **基础材质**: `M_RetroLit`
  - 禁用PBR复杂反射
  - 使用简单的Lambert漫反射
  - 硬边阴影（无过渡）
  - 纹理分辨率: 512x512 或更低
  
- **角色材质**: `M_CharacterToon`
  - 1-2级阴影分层
  - 无皮肤次表面散射
  - 纯色块风格

#### 光照策略
- 主要使用**静态光照**（烘焙）
- 少量动态光源（手电筒、路灯）
- 阴影分辨率: 1024或更低
- 雾效: Exponential Height Fog，中密度

## 二、场景设计详细规划

### 2.1 主场景区域

#### 【A】东门入口区
| 元素 | 描述 | 游戏功能 |
|------|------|---------|
| **东门斜坡** | 坑坑洼洼的水泥地，随处可见鸟屎痕迹 | 开场场景，引导玩家进入 |
| **男寝（直面东门）** | 多层建筑，外表斑驳，窗户整齐排列 | 可进入探索，收集日记 |
| **下水道** | 男寝旁，铁栅栏覆盖，散发异味 | 隐藏通道/彩蛋区域 |
| **坑坑洼洼道路** | 斜坡外的土路，雨天积水 | 外部环境延伸 |

#### 【B】校园核心区
| 元素 | 描述 | 游戏功能 |
|------|------|---------|
| **广场** | 国旗飘扬，同学们列队站立 | 关键剧情触发点 |
| **国旗杆** | 鲜艳的五星红旗（始终飘荡） | 存档点/检查点 |
| **教学楼** | "也算不上豪华"的白色建筑 | 主要室内探索区域 |
| **天光书院** | 融合古今元素的建筑 | 图书馆/档案室功能 |

#### 【C】生活服务区（黑色幽默区域）
| 元素 | 描述 | 游戏功能 |
|------|------|---------|
| **垃圾场** | 东门右侧，堆积如山 | 隐藏道具/关键线索 |
| **商店** | 垃圾场旁，靠近宿舍 | "同学们不会想到商店外面是垃圾场" - 讽刺场景 |
| **工寝** | 深处简陋建筑，招租牌子 | 工人/后勤人员区域，藏有秘密 |
| **宿舍区** | 徽派风格（官方描述） | 休息/剧情触发 |

### 2.2 场景细节清单

#### 清晨时段特有元素
- [ ] 刷牙的水杯和牙刷（宿舍阳台）
- [ ] 还没完全清醒的同学NPC
- [ ] 晨光透过国旗的效果
- [ ] 广场广播体操音乐（千禧年风格）

#### "唱戏"校长场景
- [ ] 临时搭建的演讲台
- [ ] 麦克风（带啸叫效果）
- [ ] 校长模型（更换不同服装版本）
- [ ] 咳嗽动画/音效
- [ ] 台下同学的机械鼓掌

#### 劳动场景
- [ ] 上课时间扫地的同学
- [ ] 垃圾桶和垃圾车
- [ ] 清洁工具（扫帚、簸箕）
- [ ] 老师的训斥声（远处）

## 三、剧情框架

### 3.1 主线剧情

**第一章：拾阶而上**
- 时间线：2023年，改革后的"明德学院"
- 开场：主角收到匿名邮件，回到母校
- 场景：东门斜坡 → 男寝 → 广场
- 发现：校长更换频繁的秘密

**第二章：明德之下**
- 探索：教学楼、天光书院
- 线索：教师猝死事件的真相
- 谜题：跑操禁令的由来

**第三章：博雅暗涌**
- 深入：商店的秘密、工寝的地下室
- 揭示：30万奖金招生的内幕
- 冲突：关系户学生的遭遇

**终章：回声**
- 抉择：揭露真相/保持沉默/转学离开

### 3.2 支线任务

| 任务名 | 触发点 | 内容 |
|--------|--------|------|
| 鸟屎收集者 | 东门斜坡 | 收集不同位置的鸟屎样本 |
| 商店背后 | 垃圾场旁商店 | 调查商店与垃圾场的关系 |
| 校长的咳嗽 | 广场演讲 | 统计校长咳嗽次数 |
| 下水道探险 | 男寝旁 | 探索下水道隐藏区域 |

## 四、角色设计

### 4.1 主要角色

**主角（玩家控制）**
- 年龄：18-20岁
- 身份：前博雅学生，现为调查记者/私家侦探
- 动机：寻找某人的下落/揭示真相

**校长们（轮换Boss）**
- 校长A：爱咳嗽的中年男性
- 校长B：年轻的新派管理者
- 校长C：神秘的前任（只在闪回中出现）

### 4.2 NPC设计

| NPC类型 | 特征 | 行为 |
|---------|------|------|
| 扫地同学 | 穿着校服，手持扫帚 | 在上课时段出现，重复扫地动作 |
| 早起刷牙学生 | 牙刷、毛巾、睡眼惺忪 | 宿舍区清晨出现 |
| 商店老板 | 热情但眼神闪烁 | 会主动推销商品 |
| 神秘工人 | 工寝区域 | 沉默寡言，知道秘密 |
