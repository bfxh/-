# 1. 核心叙事架构

## 1.1 故事核心概念

**"意识的幽灵"**

玩家操控的黄瑾瑜，既是真实存在的学生，也是另一个自我的"意识投射"。两个主角互相映照：
- **玩家操控的黄瑾瑜** -> 活着的、走读的、观察者
- **"鬼"黄瑾瑜** -> 意识的、寄宿在学校的、被观察的

**核心主题**: 记忆与身份的边界模糊

---

# 2. 双主角系统 (Double Protagonist System)

## 2.1 角色架构

```
真实层 (Reality Layer)
├── 黄瑾瑜 (玩家操控)
│   ├── 身份: 走读生
│   ├── 动机: 调查学校的诡异传闻
│   └── 视角: 外部观察者
│
└── "黄瑾瑜" (意识幽灵)
    ├── 身份: 寄宿在过去某个时刻的自己
    ├── 动机: 保护/警告/引导玩家自己
    └── 视角: 内部知情者
```

## 2.2 游戏机制设计

### 切换机制 (Phase Shift)

**触发条件**:
1. **特定时间**: 每天00:00自动切换
2. **特定地点**: 7楼禁区、下水道深处、垃圾场深处
3. **特定事件**: 收集到"记忆碎片"后

**视觉表现**:
```
切换动画:
  - 屏幕闪白 (0.2秒)
  - 色彩反转 (1秒)
  - 恢复正常，但世界发生细微变化:
    ├── NPC消失或变成模糊轮廓
    ├── 墙壁出现不存在的通道
    └── 时间停滞 (钟表不动)
```

### 两个主角的差异

| 特性 | 玩家黄瑾瑜 | 意识幽灵黄瑾瑜 |
|------|-----------|---------------|
| **移动** | 正常行走/跑步 | 漂浮/穿墙 |
| **交互** | 可与物体/NPC交互 | 只能观察，无法改变 |
| **视野** | 普通视距 | 可看到"隐藏的东西" |
| **危险** | 可被追逐/攻击 | 无敌，但时间有限 |
| **存档** | 正常存档 | 切换回来时自动回退 |

## 2.3 叙事结构图

```
第一章: "我已经不会 经常想他们"
├── 玩家黄瑾瑜
│   └── 场景: 走读生视角探索
│   └── 发现: 7楼的黄牌传说
│       └── 触发: 意识切换
│           └── 幽灵黄瑾瑜
│               └── 揭示: 黄牌背后的真相
│                   └── 触发: 返回现实
└── 循环继续...

第二章: "拾阶而上"
...

终章: "他们"
├── 结局A: "回到诡异的家" (走读生结局)
│   └── 黄瑾瑜毕业，却发现自己从未离开过学校
│   └── "家" 其实是学校宿舍的7楼
│
└── 结局B: "意识的融合"
    └── 两个黄瑾瑜意识到彼此
    └── 玩家选择: 取代/共存/消散
```

## 2.4 幽灵黄瑾瑜的视觉设计

### 3D模型规格

**共享基础模型**: `SK_Ghost_HuangJinyu`
- 继承自 `SK_Player_Base`
- 材质: `M_Ghost_Hologram`

**材质效果**:
```
M_Ghost_Hologram:
  └── Shader Graph:
      ├── Base Color: Yellowish-white (0.9, 0.9, 0.7)
      ├── Opacity: 0.3-0.6 (sine wave flickering)
      ├── Emission: Soft glow 0.2
      ├── Refraction: Distortion 0.1
      └── Fresnel: Edge highlight
```

**动画**:
- 行走 -> 漂浮 (无脚步动画)
- 转头 -> 360度无限制旋转
- 交互尝试 -> 手穿过物体 (无响应)

# 3. 走读生视角场景 (Commuter Student Perspective)

## 3.1 核心设计理念

**"门外的人看门内的诡异"**

作为走读生，黄瑾瑜每天进出学校，形成独特的"边缘人"视角：
- 清晨入校，看到住校生的诡异晨间仪式
- 傍晚离校，路过7楼亮着奇怪的光
- 周末偶尔返校，学校空无一人

## 3.2 走读生专属场景

### 【Scene A】"家" -> 学校 路线图

**玩家住所设计**:
```
BP_PlayerHome (走读起点)
├── 位置: 学校外3公里居民区
├── 房间:
│   ├── SM_Bed_Large (单人床)
│   ├── SM_Desk_Study (书桌)
│   ├── SM_Window_Bedroom (窗户)
│   └── SM_Poster_Lizhi (李志海报)
├── 细节:
│   ├── 李志CD: 放在书桌上
│   ├── 日记本: 可交互，记录学校诡异见闻
│   └── 闹钟: 06:00 (走读生起床时间)
└── 出门:
    └── 触发: 骑单车/公交到学校东门
```

**清晨路线序列**:
```
06:00 - 起床 (卧室)
06:30 - 出门 (居民区街道)
07:00 - 到达东门斜坡
07:10 - 经过男寝 (看到刷牙学生)
07:15 - 穿过广场 (早操准备中)
07:20 - 到达教室
```

### 【Scene B】傍晚离校路线

**关键时间点**:
- 18:00 - 放学
- 18:30 - 途经7楼 (看到亮灯)
- 19:00 - 经过商店区 (小摊贩vs商店)
- 19:30 - 听到李志歌声 (校外)
- 20:00 - 到家

**7楼观察点**:
```
Location: 教学楼西侧
Time: 18:30-19:00

Visual Elements:
  - 7楼某扇窗户亮着微弱的黄光
  - 窗户贴有黄牌 (远距离可见轮廓)
  - 偶尔有人影闪过

Interaction:
  - 玩家可尝试接近
  - 但门禁系统阻拦
  - 触发日记记录
```

### 【Scene C】周末返校

**"空城"体验**:
```
 weekend_saturday_morning.umap:
  - NPC数量: 减少90%
  - 音效: 只有风声、远处施工声
  - 特殊事件: 
    ├── 操场上残留的学生站位痕迹
    ├── 商店关门，但灯亮着
    └── 7楼传来隐约声响
```

## 3.3 走读生视角vs住校生视角对比

| 场景 | 走读生看到 | 住校生体验 |
|------|-----------|-----------|
| **男寝阳台** | 刷牙的学生剪影 | 真实互动 |
| **7楼灯光** | 远处的黄光/黄牌 | 禁区警示/无法进入 |
| **广场** | 早操的学生队列 | 参与队列/听校长演讲 |
| **商店** | 观察商店vs小摊贩冲突 | 内部购物 |
| **7楼禁区** | 无法接近 | 可以探索 (幽灵形态) |

# 4. 7楼禁区 (The Yellow-Tagged Floor)

## 4.1 禁区设定

**位置**: 学楼顶层 (原设计为实验室，后被废弃)

**黄牌**:
```
SM_Sign_YellowWarning (警示牌模型)
├── 材质: M_Sign_Paper_Yellow
├── 文字: 
│   └── "警告: 未经许可禁止入内"
│   └── "危险区域"
│   └── "后果自负"
└── 状态: 旧、破损、字迹模糊
```

**入口封锁**:
- 铁门: SM_Door_Metal_Rusty (生锈)
- 锁链: SM_Chain_Locked
- 锁: SM_Padlock_Old

## 4.2 禁区内场景设计

### 进入条件

**现实黄瑾瑜**:
- 无法物理进入
- 只能在白天透过窗户观察
- 晚上通过幽灵形态进入

**幽灵黄瑾瑜**:
- 可以穿墙进入
- 但有时间限制 (5分钟)
- 发现关键线索

### 禁区内部布局

```
Zone_Floor7_Forbidden.umap:
├── Corridor (走廊)
│   ├── 地面: 积灰、废弃纸张
│   ├── 墙壁: 旧海报、涂鸦
│   ├── 灯光: 闪烁的荧光灯
│   └── Props:
│       ├── SM_BrokenChair
│       ├── SM_DustPile
│       └── SM_OldPoster (2010年代)
│
├── Lab_Abandoned (废弃实验室)
│   ├── SM_LabTable x10
│   ├── SM_BrokenGlass x50
│   ├── SM_ChemicalBottle x20
│   └── 关键道具:
│       └── "记忆碎片" (触发剧情)
│
└── SecretRoom (密室)
    └── 终章关键场景
    └── 揭示学校核心秘密
```

## 4.3 禁区氛围设计

**光照**:
```
Lighting Setup:
  - 主光源: 微弱荧光 (Color: 0.5,0.5,0.3)
  - 辅助: 窗外月光
  - Fog: Dense (Density: 0.8)
```

**音效**:
```
SFX_Floor7_Ambience:
  - Distant footsteps (echo)
  - Fluorescent hum (looped)
  - Occasional metallic creak
```

# 5. 李志与地下文化 (The Underground)

## 5.1 文化符号植入

**李志**作为中国地下音乐文化的代表，为游戏提供独特的时代感和文化厚度。

### 5.1.1 歌词引用

**"我已经不会 经常想他们"** (来自歌曲《他们》)

**游戏内植入**:
```
校外场景:
  Location: 学校外小路
  Time: 19:00-21:00
  
  NPC: BP_Busker_LiZhiFan
    ├── 模型: 街头歌手
    ├── 动作: 弹吉他
    ├── 音频: 播放李志歌曲片段
    │   └── SFX_Lizhi_They_Snippet
    └── 对话:
        └── "你听过李志吗?"
        └── "他唱的那些, 关于学校关于青春..."
```

### 5.1.2 李志海报与CD

**玩家房间**:
```
BP_Prop_LizhiPoster:
  - Location: 玩家家卧室墙壁
  - Material: M_Poster_Paper
  - 视觉: 李志早期演出现场照片
  - 可交互: 点击查看歌词片段

BP_Prop_LizhiCD:
  - Location: 书桌上
```

### 5.1.3 李志歌曲与剧情映射

| 李志歌曲 | 剧情映射 |
|---------|---------|
| 《他们》 | 学校里的"他们" (校长/体制) |

## 5.2 地下文化场景

# 6. 商店vs小摊贩冲突 (Store vs Vendors Conflict)

## 6.1 冲突背景

**设定**:
- 商店: 学校官方指定，需交"收成" (保护费/提成)
- 小摊贩: 校外人员，无学校许可，但便宜
- 学校利益: 保护商店收益，打压小摊贩

## 6.2 场景设计

### 【Zone F】服务区的扩张

**商店与摊贩对比**:

```
ServiceArea_Vendors.umap:

Store (Official):
  ├─ SM_Shop_Building (官方建筑)
  ├─ M_Store_Signboard (学校许可标志)
  ├─ Price: High (垄断)
  └─ NPC: BP_ShopKeeper_Subservient (顺从的店主)
      └─ 对话: "学校的决定我们没办法..."

Vendors (Underground):
  ├─ BP_VendorCart x5 (小推车)
  ├─ SM_VendorUmbrella x5 (遮阳伞)
  ├─ Price: Low (竞争)
  ├─ Time: 傍晚-夜间 (躲检查)
  └─ NPCs: 
      ├─ BP_Vendor_A (大叔)
      ├─ BP_Vendor_B (大妈)
      └─ BP_Vendor_Student (兼职学生)
          └─ 对话: "嘘，别让学校保安看到"
```

### 6.2.1 摊贩被驱赶事件

**剧情触发点**:

```
Quest: "消失的摊贩"

Phase 1:
  - 玩家连续3天傍晚看到摊贩
  - 购买廉价零食 (低价格)

Phase 2:
  - 第4天傍晚，摊贩消失
  - 地上残留: SM_Trash_Scattered (散落货物)
  - 触发: BP_Event_VendorCrackdown

Phase 3:
  - 找到摊贩藏匿点 (墙后小巷)
  - 对话解锁: 学校收取商店"收成"真相
  - 获得: "内部文件碎片" (关键道具)
```

### 6.2.2 商店后门讽刺场景强化

**原设计强化**:

```
Shop_BackDoorScene.umap:

前视角 (玩家第一次来):
  - SM_Shop_Front (光鲜门面)
  - SM_Sign_Promotion (促销海报)
  - Audio: SFX_Shop_Music (轻音乐)

后视角 (发现真相后):
  - SM_TrashPile_Back (后门堆满垃圾)
  - SM_Drain_Open (堵塞的下水道)
  - SM_Rat_x3 (老鼠)
  - Audio: SFX_Shop_Back_Flies (苍蝇嗡嗡)

对比触发:
  If Player discovers backdoor:
    - Achievement: "两个世界"
    - Journal: 
      "商店的面向变了...
       面对学生的一面是明亮整洁,
       背对学生的一面是垃圾和污秽。
       这不就是学校的缩影吗?"
```

### 6.2.3 NPC态度变化

**商店店主**:
```
BP_ShopKeeper:
  
State_Normal (玩家未发现后门):
  - 对话: 热情推销
  - "欢迎光临! 今天新到了..."

State_Nervous (玩家发现后门):
  - 对话: 眼神闪烁
  - "你...你从哪里来的?"
  - "后面没什么好看的..."

State_Revelation (任务推进后):
  - 对话: 坦白
  - "学校要我们交很多...
   不交就会被换掉...
   你看外面那些摊贩,
   学校不让他们进来..."
```
