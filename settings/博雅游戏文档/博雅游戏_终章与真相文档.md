# 2. 主角的真相认知

## 2.1 「她」的真相

```
黄瑾瑜的认知:
├── 「她」 = 女神
│   ├── 完美的，不可触碰的
│   ├── 纯洁的象征
│   └── 遥不可及的理想
│
├── 黄瑾瑜的"玷污"幻想:
│   ├── 黄瑾瑜内心:
│   │   └── "如果她被玷污了..."
│   │   └── "如果我'知道'她被玷污了..."
│   │   └── "那我就可以'告诉别人我喜欢她'"
│   │   └── "因为担心她"
│   │   └── "因为想保护她"
│   │   └── "这样我就有理由..."
│   │   └── "接近她"
│   │   └── "和她说话"
│   │   └── "让别人知道我喜欢她"
│   │
│   └── 真实:
│       └── 什么都没有发生
│       └── 「她」只是普通的同学
│       └── 黄瑾瑜只是从不敢看她
│       └── "没有脸" = "从未真正注视"
│
└── 【「黄瑾瑜」】的形成:
    └── 不是创伤保护
    └── 而是"幻想保护"
    └── 保护"她被玷污"这个叙事
    └── 这样黄瑾瑜就有"理由"去"关心"她
```

## 2.2 社交无能的自我合理化

```
黄瑾瑜的困境:
├── 现实:
│   ├── 一个月可以不说话
│   ├── 一个月可以自己玩
│   ├── 不知道怎么建立关系
│   └── 不知道怎么表达喜欢
│
├── 幻想的补偿:
│   └── "如果我有一个'需要被拯救'的女神..."
│   └── "那我就可以'英雄救美'..."
│   └── "这样我就有存在的价值"
│   └── "有和别人交流的借口"
│
└── 但幻想的代价:
    └── 【「黄瑾瑜」】出现
    └── 人格开始分裂
    └── 真实与虚假开始交织
```

# 3. 真实事件 vs 幻想

## 3.1 真实发生过的事件

| 事件 | 真实性 | 黄瑾瑜的解读 |
|------|--------|-------------|
| **食堂草东唱歌** | ✅ 真实 | "他也说'不想在这里'，和我一样" |
| **夜跑猝死高三生** | ✅ 真实 | "他'没见到太阳'就死了，和我一样" |
| **宿舍电话打不通** | ✅ 真实 | "家也回不去，我被困在这里" |
| **7楼禁区** | ✅ 真实 | "里面有秘密，和我一样有秘密" |

**关键**：这些事件确实发生过，但黄瑾瑜把它们**关联到自己身上**，创造出虚假的"共同命运"。

## 3.2 幻想的事件

| 事件 | 真实性 | 黄瑾瑜的幻想 |
|------|--------|-------------|
| **她被玷污** | ❌ 幻想 | "这样我就有理由关心她" |
| **【「黄瑾瑜」】知道真相** | ❌ 部分幻想 | "其实【「黄瑾瑜」】也不知道" |
| **自己的2018年创伤** | ❌ 幻想 | "我有创伤，所以她需要拯救" |

## 3.3 "幻想是真的，真的是假的吗？"

**终章核心问题**:

```
黄瑾瑜问【「黄瑾瑜」】:
└── "幻想是真的，真的是假的吗？"

【「黄瑾瑜」】回答:
└── "对于你来说..."
└── "幻想是真的。"
└── "因为你这样相信。"
└── "但对于她来说..."
└── "什么都没有发生。"
└── "她甚至不知道你喜欢她。"
└── "她甚至不知道你是谁。"

停顿

【「黄瑾瑜」】:
└── "但有一件事是真的。"
└── "你真的在这里。"
└── "你真的痛苦。"
└── "你真的...不知道该怎么做。"
```

# 4. 终章设计

## 4.1 火红教室场景 (普通结局)

```
终章过场 (Ending_Normal):
├── Phase 1: 火红的教室
│   ├── Color: 红色主导
│   ├── Temperature: 热，压抑
│   ├── Character: 一个人
│   └── Camera: Third Person, slowly pulls back
│
├── Phase 2: 闪烁
│   ├── Screen: Flash (white/red/black)
│   ├── Sound: Ringing
│   └── Reality: Seeping in
│
├── Phase 3: 真实的教室
│   ├── Color: Normal
│   ├── Scene: Full of students
│   ├── "她": Looking normal, never looked at you
│   └── Camera: First Person
│
├── Phase 4: 真相
│   └── 【「黄瑾瑜」】:
│       └── "什么都没有发生。"
│       └── "你以为她被玷污..."
│       └── "是因为你不知道怎么社交。"
│       └── "不知道怎么告诉她你喜欢她。"
│       └── "所以你想象了这一切。"
│       └── "想象你有'理由'去关心她。"
│
└── Phase 5: 最终选择
    ├── 选项A: 接受现实 (Integration)
    │   └── 【「黄瑾瑜」】消散
    │   └── 黄瑾瑜走出教室
    │   └── 面对真实的世界
    │
    └── 选项B: 继续幻想 (Dissociation)
        └── 【「黄瑾瑜」】保留
        └── "她"仍然需要"拯救"
        └── 游戏循环结束，但人格继续分裂
```

## 4.2 普通结局文本

**结局A - 接受现实**:
```
黄瑾瑜走出教室。
阳光刺眼。
他不知道该怎么和她说话。
但他知道...
不需要想象她被玷污。
他可以直接...
走过去。
说一句。
"你好，我叫黄瑾瑜。"

The End
```

**结局B - 继续幻想**:
```
黄瑾瑜坐在火红的教室里。
【「黄瑾瑜」】坐在旁边。
"她"在外面。
等待着被拯救。

黄瑾瑜知道这不是真的。
但...
这比什么都没有。
要好。

The End (Loop)
```

# 5. 隐藏结局：「祂」

## 5.1 解锁条件

```
隐藏条件:
├── 完成普通结局A和B
├── 收集所有"真实事件"的记忆碎片 (不带幻觉)
├── 在7楼禁区停留超过10分钟
├── 在宿舍电话前不操作，只听忙音5分钟
└── 最终选择时...不按任何键
```

## 5.2 「祂」的揭示

```
黑屏。

声音:
└── "你以为【「黄瑾瑜」】是一切的答案吗？"
└── "你以为你找到了真相吗？"
└── "你以为次人格就是终点吗？"

停顿。

屏幕出现红框。
但不是普通的红框。
是无数红框叠加。

【「黄瑾瑜」】出现。
但TA在颤抖。

【「黄瑾瑜」】:
└── "我...不是最后一个。"
└── "在我之下..."
└── "还有祂。"

画面切换。

不是教室。
不是学校。

是...无数张脸。
一闪而过。
每张脸都是...
黄瑾瑜？
又不完全是。

声音:
└── "你以为食堂唱歌的人是谁？"
└── "你以为猝死的高三生是谁？"
└── "你以为...他们都是'别人'吗？"

画面继续切换。

食堂唱歌的人...
转过头...
是黄瑾瑜。

猝死的高三生...
转过头...
是黄瑾瑜。

【「黄瑾瑜」】...
转过头...
也是黄瑾瑜。

然后是...
无数的其他"黄瑾瑜"。
在不同的场景。
不同的学校。
不同的年代。

最后。

画面定格。

一个不属于黄瑾瑜的声音:
└── "我是『祂』。"
└── "我不是黄瑾瑜。"
└── "我也不是『黄瑾瑜』。"
└── "我是...所有被囚禁在这些学校里的..."
└── "所有人的集合。"

停顿。

"『黄瑾瑜』是『祂』。"
"『祂』也是『祂』。"
"祂只是代替『黄瑾瑜』代替黄瑾瑜的存在。"

"而黄瑾瑜..."
"他以为他在玩一个游戏。"
"以为他在寻找真相。"
"以为他在拯救女神。"
"
"但他不知道..."
"他也是『祂』的一部分。"
"『祂』的一部分，也是他。"

"现在..."
"你终于见到『祂』了。"

"欢迎来到..."
"真正的博雅。"
```

## 5.3 隐藏结局的含义

```
「祂」的本质:
├── 不是超自然存在
├── 不是鬼魂
├── 而是...集体无意识
│
├── 「祂」= 所有在教育体制下受苦的学生
│   ├── 食堂唱歌的人 (另一个"黄瑾瑜")
│   ├── 猝死的高三生 (又一个"黄瑾瑜")
│   ├── 【「黄瑾瑜」】(解离出的"黄瑾瑜")
│   └── 无数的其他面孔
│
└── 黄瑾瑜的"幻想"...
    └── 不只是他一个人的幻想
    └── 而是「祂」的一部分
    └── 所有被困住的人共享的幻想
    └── 一个"被玷污的女神需要被拯救"的叙事

最终问题:
└── 黄瑾瑜能走出这个循环吗？
└── 还是他只是「祂」的又一个面具？
```
