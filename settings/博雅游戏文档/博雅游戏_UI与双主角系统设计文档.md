# 1. 双重人格系统核心设计

## 1.1 角色架构 (主人格 / 次人格)

```
主人格层 (Host Personality)
├── 黄瑾瑜 (玩家操控的主体)
│   ├── 身份: 走读生，2024年
│   ├── 记忆: 缺失2018年之前的大部分记忆
│   ├── 认知: 不知道次人格存在 (初期)
│   └── 状态: 相对稳定，但逐渐崩溃
│
└── 「黄瑾瑜」 (Dissociated Sub-Personality)
    ├── 身份: 寄宿在2018年的创伤人格
    ├── 记忆: 完整的2018年发生了什么
    ├── 动机: 保护主人格不被真相击垮
    └── 状态: 危险，充满攻击性/保护欲
```

## 1.2 人格关系图

| 维度 | 黄瑾瑜 (主人格) | 「黄瑾瑜」 (次人格) |
|------|----------------|---------------------|
| **时间** | 现在 (2024) | 停留在创伤时刻 (2018) |
| **状态** | 活着，逐渐失忆 | 创伤记忆的具象化 |
| **知情权** | 刻意遗忘过去 | 知道全部真相 |
| **动**
| **口头禅** | "你在哪？" | "我一直都在。" |

## 1.3 共存机制详解

### 人格切换状态

```cpp
enum class EPersonalityState
{
    HOST_ONLY,          // 只有主人格 (正常状态)
    CO_PRESENCE,        // 共存 (次人格旁观)
    INTRUSIVE,          // 侵入 (次人格插话/幻觉)
    SWITCH,             // 切换 (次人格控制身体)
    FUSION,             // 融合 (终章选择)
    CONFLICT            // 人格冲突 (挣扎)
};
```

### 触发条件

| 状态 | 触发条件 | 表现 |
|------|---------|------|
| **HOST_ONLY** | 游戏开场 | 正常游玩 |
| **CO_PRESENCE** | 特定地点/时间 | 次人格出现(只有黄瑾瑜一个人能看到) |
| **INTRUSIVE** | 接近创伤记忆线索 | 红框出现 + 插入对话 + 幻觉 |
| **SWITCH** | 进入7楼禁区/高压力场景 | 玩家不知情地"失去时间"，次人格控制 |
| **FUSION** | 终章 | 两格人格合为一体 |
| **CONFLICT** | 关键抉择时刻 | UI抖动+红蓝分屏+人格争夺 |

### 【特殊触发 A】早读教室场景

**触发条件**: 黄瑾瑜回到班级，早读时间

**场景表现**:
```
Blackboard_TV_Display:
├── PPT内容:
│   ├── "不准不拼搏"
│   ├── "不准不努力"
│   ├── "不准不学习"
│   └── "拼搏到无能为力，努力到感动自己"
├── 背景色: #FF0000 (鲜艳红色)
├── 字体: 加粗黑体，白色
└── 滚动播放 + 音效

Teacher_Behavior (班主任):
├── 动作: 巡视班级，拍照
├── 拍照Flash: 全屏白光 (0.2s)
├── 音频:
│   ├── "大声点！"
│   ├── "听不见！"
│   └── 试来试去的声音
└── 学生反应: 机械式大声朗读

Student_Behavior:
├── 早读状态:
│   ├── 声音: 机械、无感情、高分贝
│   ├── 表情: 空洞、无表情
│   └── 动作: 同步动作
├── 班主任拍照时:
│   ├── 突然更大力朗读
│   └── 姿态更"积极"
└── 其他班:
    └── 同样场景，声音交织
```

**次人格出现时机**:
```
当黄瑾瑜在早读教室中:
├── 班主任第3次拍照后
├── 次人格 「黄瑾瑜」 出现(只有玩家可见)
│   ├── 位置: 屏幕右侧或教室角落
│   ├── 外观: 模糊人形
│   ├── 红框: 屏幕边缘红色闪烁边框
│   └── 对话:
│       └── "你想起来了吗？"
│       └── "这就是我们的日常。"
│       └── "快点离开。"
└── 触发条件:
    ├── 主人格停留超过30秒
    └── 主人格尝试与同学对话
```

### 【特殊触发 B】食堂 — 「草东记忆」

**触发条件**: 黄瑾瑜经过食堂门口 或 进入食堂内部

**场景设计**:
```
食堂内部 (Cafeteria_Inside.umap):
├── Visual:
│   ├── 长条餐桌 (SM_Table_Cafeteria)
│   ├── 学生们低头吃饭 (无表情，机械咀嚼)
│   ├── 角落有人抱着吉他 (BP_Guitar_Player)
│   └── 周围人对弹唱者无反应
│
├── Audio:
│   ├── 底色: 餐具碰撞声、咀嚼声 (低音量)
│   └── 突兀: 吉他弹唱《但》(草东没有派对)
│       ├── "你说你不想在这里"
│       ├── "我也不想在这里"
│       ├── 音量: 不大，但清晰可辨
│       └── 音质: 轻微失真，像是从记忆里传来
│
├── 记忆的真实感:
│   └── 【「黄瑾瑜」】的旁白:
│       └── "有人在食堂唱过他的歌。"
│       └── "只有一次。"
│       └── "我现在已经不知道他是谁了。"
│       └── "或许...只有唱了这一次吧。"
│   └── 弹唱者的脸: 模糊，无法辨认
│   └── 周围人: 毫无反应，继续吃饭
│
├── NPC反应:
│   ├── 周围学生: 无反应，继续吃饭
│   ├── 食堂阿姨: 看一眼，不管，继续工作
│   └── 黄瑾瑜: 只有你能注意到这个弹唱者
│
└── 次人格触发:
    └── 当你盯着弹唱者超过5秒:
        ├── 红框闪烁 (红色边框，2Hz频率)
        ├── 【「黄瑾瑜」】出现:
│       └── "只有一次..."
│       └── "我现在已经不知道他是谁了。"
│       └── "或许，只有唱了这一次吧。"
        └── 弹唱者人影淡出消失

技术规格:
BP_Cafeteria_Memory:
├── TriggerVolume: 食堂门口走廊 + 食堂内部
├── Timeline:
│   ├── 0s: 远处吉他声起 (草东《但》片段)
│   ├── 5s: 次人格红框出现
│   ├── 10s: 人影消失
│   └── 15s: 声音渐远，记忆碎片解锁
└── Reference Audio: 草东没有派对 - 《但》
    └── "你说你不想在这里"
```

## 2.2 风格指南

### 视觉风格定位

**"千禧年CRT + 赛博朋克数据流"**

**核心元素**:
- 像素字体 + 扫描线
- CRT弯曲效果
- 数据流/Glitch特效
- 蓝/黄色调 (V) vs 红/黑色调 (银手)

### 设计规范

```css
/* 字体规格 */
Font_Pixel_Mono:  "MS Sans Serif" | "Zpix" | "VT323"
  - 大小: 8pt-14pt
  - 抗锯齿: None (Pixel Perfect)

Font_Dialogue:    "Noto Sans SC" (中文字体, 带轻微噪点)
  - 大小: 14pt-16pt
  - 字间距: 0.5em

/* 色彩系统 */
Color_V (黄瑾瑜/现实):     #1E3A5F (深蓝) + #4A90E2 (亮蓝)
Color_Silver (银手/精神):  #8B0000 (暗红) + #FF4500 (橙红)  
Color_Neutral:              #C0C0C0 (扫描线灰)
Color_Warn:                 #FF0000 (警告红)
Color_Ghost:                #FFD700 (幽灵金)

/* 特效参数 */
Scanline_Density:  2px 间距
Glitch_Amount:     偶尔抽搐
CRT_Curve:         边缘弯曲 0.05
Ghost_Opacity:     0.3-0.6 浮动
```

// WBP_SilverHand_Vision

Components:
  - Material: M_Vision_Scan

  - Appear: 0.3s fade in
  - Hold: 2s steady
  - Disappear: 0.3s fade out + glitch

# 6. 终章场景设计

## 6.1 终章概述

**目的地**: 7楼禁区最深处  
**目标**: 完成银手的未竟之事  
**结局分歧**: 4种不同终局

```
终章流程:

Chapter_Final.umap
├── Entering Floor 7
│   └── 现实黄瑾瑜最后一次进入现实7楼
│
├── The Merge (融合)
│   └── 两个黄瑾瑜意识到必须面对过去
│   └── 玩家选择: 谁主导?
│
├── The Destination (目的地)
│   └── 禁区密室
│   └── 揭示: 银手是谁/为何被困
│
└── Final Choice (最终抉择)
    ├── Ending_A: "我留下" (银手结局)
    ├── Ending_B: "我带走" (逃离结局)
    ├── Ending_C: "我们一起" (融合结局)
    └── Ending_D: "从未发生" (遗忘结局)
```
