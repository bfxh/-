# 2. 角色3D资产规格

## 2.1 通用角色技术规格

### 2.1.1 模型规格

| 项目 | 主角 | NPC（动态） | NPC（静态/操场） |
|------|------|-------------|------------------|
| **面数** | 3000-5000 tris | 2000-3000 tris | 500-1000 tris |
| **骨骼数量** | 55-65 bones | 45-55 bones | 15-20 bones (简化) |
| **UV通道** | 2 (Diffuse + Lightmap) | 2 | 1 |
| **LOD级别** | 3级 | 2级 | 无LOD（始终低模） |
| **绑定** | 标准UE5人形骨骼 | 共享骨骼 | 简化骨骼 |

### 2.1.2 纹理规格

| 贴图类型 | 主角 | NPC（动态） | NPC（静态） | 格式 |
|---------|------|-------------|-------------|------|
| **Base Color** | 512x512 | 256x256 | 256x256 | TGA/PNG |
| **Normal Map** | 512x512 | 不使用 | 不使用 | TGA |
| **ORM Map** | 512x512 | 不使用 | 不使用 | TGA |
| **压缩** | BC7 | BC3 | BC1 | DDS |

**注意**: 千禧年风格不需要高精度纹理，故意使用低分辨率

### 2.1.3 材质配置

```cpp
// M_Character_Master 材质节点结构

Base Color = Texture Sample (512x512, 无过滤/Point Sampling)
    ↓
Multiply by Color Parameter (可调校服颜色)
    ↓
Clamp Saturation (0.7-0.9) // 降低饱和度
    ↓
[Cel Shading Node]
    - Light Dot Normal
    - Step Function (0.5, 0.8) // 2级阴影
    - Hard Edge (无渐变)
    ↓
Final Output
```

## 2.3 NPC规格

### 2.3.1 动态NPC (扫地学生、刷牙学生等)

**共享骨骼网格**: `SK_NPC_Base`
- 面数: 2500 tris
- 使用变体材质实例改变外观

**变体材质实例**:
```
MI_Student_Uniform_A (蓝色校服)
MI_Student_Uniform_B (白色校服)
MI_Student_Janitor (清洁服装)
```

**AI行为树结构**:
```
BT_Student_Clean (扫地学生)
├── Sequence
│   ├── Find Patrol Point
│   ├── Move To Point
│   ├── Play Sweeping Animation
│   └── Wait (Random 3-5s)
└── Loop
```

### 2.3.2 静态NPC (操场学生 - 性能优化)

**优化策略**:
1. 使用**实例化静态网格体** (Instanced Static Mesh)
2. 无骨骼动画，使用**顶点动画纹理**(VAT) 实现微动
3. 单一材质，批处理渲染

**技术实现**:
```cpp
// BP_Student_Static_Instancer

Component: Hierarchical Instanced Static Mesh
  - Static Mesh: SM_Student_LOD0
  - Material: M_Student_Static_VAT
  - Instance Count: 200 (操场学生数量)
  - Culling Distance: 3000 units
  - Cast Shadows: False
```

**VAT材质节点**:
```
World Position Offset:
  - Vertex Animation Texture
  - Slight Sway (0.5cm amplitude, 2s period)
  - Randomize per instance (Instance Random node)
```

- 特殊动画:
  - `Anim_Speech_Loop` (演讲循环)

**行为树**:
```

├── Move to Podium

│   └── Play Cough Montage + Audio
└── Loop
```
